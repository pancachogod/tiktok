Subasta TikTok LIVE

Esta es la interfaz web que acompaña tu subasta de donadores para TikTok LIVE. La aplicación está construida con Streamlit y se ejecuta localmente; aquí encontrarás las instrucciones para ponerla en marcha y exponer el overlay que usa TikTok Studio.

1. Ejecuta la aplicación

En una terminal, clona el repositorio y arranca el servidor de Streamlit. Asegúrate de tener un entorno de Python 3.9 o superior con las dependencias instaladas. 

git clone https://github.com/pancachogod/tiktok.git
cd tiktok
pip install -r requirements.txt
streamlit run app.py --server.address 0.0.0.0 --server.enableCORS false

El servidor quedará disponible en http://localhost:8501. Si lo ejecutas en otra máquina, sustituye localhost por la IP o dominio accesible desde tu estudio.

2. Configura el overlay para TikTok Studio

Paso 1: Abre la aplicación Streamlit y, en la barra lateral, busca el bloque “Overlay TikTok Studio”.
Paso 2: Introduce la URL base que verá TikTok Studio (por ejemplo http://localhost:8501 o http://TU_IP_LOCAL:8501). El sistema generará automáticamente el enlace con el parámetro ?overlay=1.
Paso 3: Copia el enlace generado y pégalo en una fuente de navegador (Browser Source) dentro de TikTok Studio. Ajusta el tamaño para mostrar el contador y el ranking en pantalla.

3. Modo overlay

El modo overlay elimina la barra lateral y el encabezado de Streamlit, dejando únicamente el contador y el Top 3 de donadores. Recarga automática cada segundo mientras el temporizador esté activo o lleguen eventos desde TikTok.

Si necesitas un enlace rápido para pruebas locales, abre http://localhost:8501/?overlay=1 una vez que la aplicación esté corriendo.
